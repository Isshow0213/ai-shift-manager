# AI Shift Manager

## 概要

AI Shift Manager は、アルバイトのシフト希望提出と店長のシフト作成を効率化するためのWebアプリです。

従業員はシフト希望を提出し、管理者は必要人数を設定した上で、自動生成されたシフトを確認できます。

## 解決したい課題

現在、シフト希望はLINEなどで個別に送られることが多く、店長は提出状況の確認やシフト作成に多くの時間を使っています。

このアプリでは、シフト希望の回収とシフト作成をWeb上で一元管理することで、店長の負担を減らすことを目指します。

## 現在実装済みの機能

- ログイン機能
- カスタムUserモデル
- シフト希望提出
- 提出済みシフト希望一覧
- 必要人数設定
- 確定シフト保存
- 簡易シフト自動生成
